# Use jira api to create a jira issue. 
# 1. Finish step 2 at https://developer.atlassian.com/cloud/jira/software/getting-started/
# 2. Create api token as per https://confluence.atlassian.com/cloud/api-tokens-938839638.html
# 3. In postman, setup environment variable {{JiraHost}}
# 4. Add new request as:
#    Method: POST
#    URI: https://{{JiraHost}}/rest/api/3/issue
#    Authentication: Basic Authentication
#        username: <your email address as atlassian login id>
#        password: <your api token>
#        body: <sample as below>
{

                    "type": "paragraph",

                            "text": "This is a test."
                        }

            ]
